## Install

npm:
```
npm install ghost-kernel
```

bower:
```
bower install ghost-kernel
```

## Usage

node.js(ES2015):
```javascript
import {Ghostkernel} from 'ghost-kernel';
```

node.js(ES5):
```javascript
var ghostKernel = require('ghost-kernel');
var Ghostkernel = ghostKernel.Ghostkernel;
```

browser:
```html
<script src="ghost-kernel.js"></script>
var Ghostkernel = ghostKernel.Ghostkernel;
```
